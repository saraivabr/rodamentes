### A ANATOMIA DA TEIA DE OFERTAS

A Teia de Ofertas não é uma tática. É uma **arquitetura de monetização** onipresente onde múltiplos caminhos levam ao seu produto principal.

Quando a pessoa está dentro da TEIA (INSTAGRAM) \- vários caminhos diferentes a levam inevitavelmente à venda:

#### CAMINHO 1: STORY → PRODUTO PRINCIPAL (DIRETO)

```
┌─────────────┐     ┌─────────────────────┐
│             │     │                     │
│  STORY      │────►│  PRODUTO PRINCIPAL  │
│  LEVANTADA  │     │                     │
│  DE MÃO     │     │                     │
└─────────────┘     └─────────────────────┘
```

**Como funciona:**

1. Cria story com call-to-action direto ("deslize para cima")  
2. Leva diretamente para página de vendas ou aplicação do produto principal  
3. Ideal para ofertas com escassez real (vagas limitadas/tempo limitado)

#### CAMINHO 2: LINK DA BIO → VÍDEO → APLICAÇÃO → PRODUTO PRINCIPAL

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────────────┐
│             │    │             │    │             │    │                     │

│             │    │             │    │             │    │                     │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────────────┘
```

**Como funciona:**

1. Anuncia vídeo especial/exclusivo no link da bio  
2. No vídeo, cria desejo e explica transformação  
3. Direciona para formulário de aplicação que qualifica prospects  
4. Aprova candidatos ideais para o produto principal

#### CAMINHO 3: ÁREA DE MEMBROS → ENTREGA PRODUTO 1 → PITCH PRODUTO PRINCIPAL

```
┌─────────────┐    ┌─────────────┐    ┌─────────────────────┐

```

**Como funciona:**

1. Cliente acessa área de membros de produto que já comprou  
2. Ao final de cada módulo/conteúdo, há recomendações personalizadas  
3. Seção "Próximos Passos" direciona para produto principal como evolução natural

#### CAMINHO 4: POST DEMONSTRAÇÃO → DOC → PRODUTO ENTRADA → PRODUTO PRINCIPAL

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────────────┐

```

**Como funciona:**

1. Compartilha resultados reais (seus ou de clientes) em post do feed  
2. Oferece documento gratuito explicando o método por trás dos resultados  
3. No documento, oferece produto de entrada acessível para primeiros passos  
4. Dentro do produto de entrada, escalona para produto principal

#### CAMINHO 5: PRODUTO ENTRADA → PEDE DEPOIMENTO → PRODUTO PRINCIPAL

```
┌─────────────┐    ┌─────────────┐    ┌─────────────────────┐
│             │    │             │    │                     │
│  PRODUTO    │───►│  FORMULÁRIO │───►│  PRODUTO PRINCIPAL  │

**Como funciona:**

1. Cliente implementa e obtém resultado com produto de entrada  
2. Solicita depoimento através de formulário específico  
3. Após enviar depoimento, recebe oferta exclusiva para produto principal  
4. Oferta posicionada como "próximo nível lógico" para quem já teve sucesso

#### CAMINHO 6: DESTAQUE CASOS → DM → FORMULÁRIO → PRODUTO PRINCIPAL

```

**Como funciona:**

1. Cria destaque específico com casos de sucesso/transformações  
2. No final, pede para enviar palavra-chave específica na DM  
3. Resposta automática pede informações básicas e envia link de formulário  
4. Formulário qualifica e direciona para oferta do produto principal

#### CAMINHO 7: REELS → CAIXINHA → RESPOSTA → PRODUTO PRINCIPAL

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────────────┐

```

**Como funciona:**

1. Cria Reels educacional ou de entretenimento com alto potencial viral  
2. Nos stories seguintes, abre caixinha de perguntas sobre o tema do Reels  
3. Seleciona perguntas estratégicas para responder que abrem porta para solução  
4. Na resposta, menciona produto principal como solução completa

#### CAMINHO 8: CARROSSEL TENSÃO → PERFIL → HIGHLIGHTS → PRODUTO PRINCIPAL

└─────────────┘    └─────────────┘    └─────────────┘    └─────────────────────┘

1. Publica carrossel que cria tensão psicológica sem resolver completamente  
2. Prospect curioso visita seu perfil para buscar mais informações  
3. Highlights estratégicos chamam atenção mostrando método/sistema  
4. Highlight do método leva diretamente ao produto principal
